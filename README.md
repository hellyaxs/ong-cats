## 🚀 Instalando Ong de gatinho 

Para instalar o Ong de gatinho , siga estas etapas:

Linux, macOS Windows:

```bash
git clone https://github.com/hellyaxs/projeto-KBRTEC.git && cd projeto-KBRTEC
```

```bash
npm install
```

docker:

```bash
docker build -t app:latest . 
```
